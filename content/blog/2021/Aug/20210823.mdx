Here are some notes on Using Material-UI  

create a new app "materialui" from the standard template:
```terminal
npx create-react-app materialui

```

That will create the folder 'materialui' - open that in VS Code...  

Next, let's add some materia-ui parts:

```terminal
npm i @material-ui/core
npm i @material-ui/icons

```

notes:

```javascript
import React, { useState } from 'react';

function Example() {
  // Declare a new state variable, which we'll call "count"
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
```
